How to Hexo

2019-01-31

1. Membuat Akun dan Reporsitory Github

Buat akun Github setelah mendaftar,buat sebuah Reporsitory dengan cara:

####New Reporsitory > isikan nama Reporsitory dengan nama : (username kalian).github.io >set Public > Create Reporsitory.####

2. Install Node.JS

Jadi pastikan terlebih dahulu apakah kalian sudah menginstall nodejs dan npm atau belum. kalian bisa mengechecknya diterminal dengan command

Jika sudah,kalian bisa mengecheck Hexo kalian dengan commad

Jika sudah,akan muncul versi Hexo kalian.

3. Membuat github pages

4. Memilih theme

Buka website Hexo Theme dan download salah satu tema yang kalian inginkan

pada bagian theme, isikan dengan nama tema kalian.
